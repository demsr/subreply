About Emoji

{{ sub.full_name | emojize }} was created by {{ luc.full_name | emojize }} from the desire of having a simple to use, English-only, global and public forum that has nothing in common with ancient and untrustworthy social networks.

Register your account, improve your profile, wait for approval then see for yourself if it's worthy of your time. You can change your mind at any time.

Built around user stories

  1. a user starts interesting threads
  2. a user writes witty replies
  3. a user reads quality news articles
  4. a user discovers highlighted or saved content
  5. a user reads or writes long stories (soon)

Features and limitations

  1. reverse-chronological order for all feeds
  2. unique name handles like {{ emo.full_name | emojize }}
  3. light and dark themes based on OS preference
  4. 640 characters per reply
  5. ASCII and emojis for content
  6. full thread and entire conversation support
  7. only one link, mention or hashtag per reply
  8. no support for paragraphs, keeps conversation tidy
  9. no popularity counters or instant gratification

Keeping things private

  • edit or delete your replies before they are replied to
  • delete received replies before they are replied to
  • no redistribution of data, private or public based on GDPR
  • email address needed for password reset, no spam
  • straightforward and complete account deletion
  • data export in a human readable format
  • logparser code for parsing access logs is open-sourced

Technology stack

  • high frequency Vultr VPS running Ubuntu LTS
  • coded in Python using Falcon framework and Django ORM
  • Gunicorn in front of Nginx, PgBouncer in front of PostgreSQL
  • SVG icons by Heroicons, Route 159 font by dotcolon.net
