About

{{ sub.full_name }} was created by {{ luc.full_name }} from the desire of a having a simple to use, English-only, public forum that has nothing in common with ancient and untrustworthy social networks.

Register your account, improve your profile, wait for approval then see for yourself if it's worthy of your time. You can change your mind at any time.

Built around user stories

  1. an user comes to start good threads
  2. an user comes to write witty replies
  3. an user keeps in touch with private messages
  4. an user reads quality news articles
  5. an user discovers highlighted and saved content

Features and limitations

  1. reverse-chronological order for all feeds
  2. unique name handles like {{ emo.full_name }}
  3. mentions, links, hashtags and hashrefs support
  4. light and dark themes based on OS preference
  5. 640 characters per reply
  6. ASCII and emojis for content
  7. full thread and a full conversation views
  8. only one bang, mention, link or hashtag per reply
  9. no support for paragraphs, keeps conversation tidy
  10. no popularity counters or instant gratification

Keeping things private

  • edit or delete your replies before they are replied to
  • delete received replies before they are replied to
  • private and server-side encrypted messages
  • no redistribution of data, private or public based on GDPR
  • email address needed for password reset, no spam
  • straightforward and complete account deletion

Technology stack

  • high frequency Vultr VPS running Ubuntu LTS
  • coded in Python using Falcon framework and Django ORM
  • Gunicorn in front of Nginx, PgBouncer in front of PostgreSQL
  • SVG icons by Heroicons, Route 159 font by dotcolon.net
